# C Library

> یک کتابخانه استاندارد و کاربردی در زبان C شامل توابع ریاضی و رشته‌ای برای مصارف عمومی

---

# 📚 فهرست توابع

## توابع ریاضی

| توضیحات | تابع |
|---------|------|
| بررسی عدد اول بودن | `isPrime(int n)` |
| محاسبه فاکتوریل | `factorial(int n)` |
| معکوس کردن عدد | `reverseNumber(int n)` |
| محاسبه مجموع ارقام | `sumDigits(int n)` |
| شمارش تعداد ارقام | `countDigits(int n)` |
| محاسبه بزرگترین مقسوم‌علیه مشترک (ب.م.م) | `gcd(int a, int b)` |
| محاسبه کوچکترین مضرب مشترک (ک.م.م) | `lcm(int a, int b)` |
| بررسی فرد بودن | `isOdd(int n)` |
| بررسی زوج بودن | `isEven(int n)` |

## توابع رشته‌ای

| توضیحات | تابع |
|---------|------|
| معکوس کردن رشته | `reverseString(char str[])` |
